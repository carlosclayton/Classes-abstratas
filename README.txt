Classes abstratas são utilizadas quando precisamos deixar certos detalhes de implementação para nossos herdeiros. Especialmente quando precisamos criar objetos cujos os métodos não são totalmente definidos. Com o uso de abstração, podemos criar uma arquitetura muito extensível dentro de nossos projetos de desenvolvimento.
