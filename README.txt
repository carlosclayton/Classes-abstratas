Classes abstratas 
=================
As Classes abstratas são utilizadas quando precisamos deixar certos detalhes de implementação para nossos herdeiros, especialmente quando precisamos criar objetos cujos os métodos não são totalmente definidos. Através do uso de abstração, podemos criar uma arquitetura muito extensível dentro de nossos projetos de desenvolvimento.
Estes códigos fazem parte do artigo Classes abstratas em PHP disponível no link: http://www.viusolucoes.com.br/blog-viu-solucoes/categoria/php/classes-abstratas-em-php
